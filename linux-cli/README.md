### Terminals, Commands and Shells (Oh My!)

- Command = 文字, case-sensitive
- Command + Enter = Shell 解譯
  - gift -> English -> 禮物
  - gift - German -> poison
  - smoking - English -> 抽菸
  - smoking - French -> 西裝
- shell type
  - bash 最常見, Linux, Ubuntu
- terminal: shell 之窗
